ABOUT

This website is the interface for the Kumbh Mela (meta)database with which we try to keep track of all the incoming files for the various subexperiments during the Kumbh Mela festival in April/May 2016. See this link for more information on the overall project.

The layout for this website is taken from free-css.com. The icons used to represent the various forms of data were taken from the Font Awesome collection.

The code is publicly available at https://github.com/louisdijkstra/kumbhmela-metadata under the MIT license.
